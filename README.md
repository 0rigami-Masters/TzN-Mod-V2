# TzN Mod V2

**TzN Mod** is a custom Venge.io mod/overlay designed around competitive play, clan identity, and UI customization.

Built for the **TzN** brand, this project adds custom styling, menu/game enhancements, and settings that help create a cleaner and more distinctive in-game experience.

## Features

- **TzN clan tag customization**
- **Animated UI effects**
- **Settings For TzN Mod**
- **Twitch Companion**
- **Premium black/gold themed branding**
- **Reminder Blocker**
- **Custom Game Audio**
- **Enemy Health Numbers**
- **Game Visual Improvements**
- **Join Link And Spectate Features**

## Keybinds
- F1 - Open Spectate Link from "https://venge.io/#XXXX" Format (CLIPBOARD)
- F2 - Copy Match Link
- F3 - Toggle Hide/Show Weapon And Hands
- F4 - Refresh to Lobby
- F5 - Refresh current link
- F6 - Join Link from "https://venge.io/#XXXX" Format (CLIPBOARD)
- F7 - Venge Client Settings
- F8 - TzN Mod Settings
- F11 - Full Screen

## Overview

This project was created to give TzN a more exclusive, polished, and recognizable visual presence in-game while keeping the mod practical for everyday use.

The goal is to combine:
- clean presentation
- competitive identity
- quality-of-life improvements
- flexible customization
